# discord s/i/x-seven hack

six seven in discord.

[example video](https://github.com/user-attachments/assets/6b8caa86-7211-4f07-a460-990facaebcb2)

## how this works

the discord client parses `sed` commands and runs them on your most recent message in the chat; this means that running `s/i/x-seven` replaces the first instance of the letter `i` in your last message with `x-seven`. 
for example, if your previous message included a link to a gif of beloved actor chris pratt as mario (`https://klipy.com/gifs/its-me-mario-nintendo`), it would be edited to `https://klx-sevenpy.com/gifs/its-me-mario-nintendo`.

thus, if someone bought the domain `klx-sevenpy.com`, they could write a program that fetches the first frame of the original gif, splits it in half, animates the two halves going up and down, and overlays an impact font caption reading "s/i/x-seven !!!". that's exactly what i did, and you're looking at the code i wrote to do that.

## usage

1. send a gif from the klipy gif picker
2. send a message containing `s/i/x-seven`
3. six seven in discord.
